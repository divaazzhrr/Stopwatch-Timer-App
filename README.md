# Stopwatch & Timer Apps

## Description
Simple stopwatch & timer app built with vanilla JavaScript to help track time easily.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Stopwatch
- Timer
- Dark mode toggle

## Setup Instructions
1. Clone this repository
2. Open `index.html` in your browser
